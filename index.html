HTML

  • The head element contains information about the webpage, and is invisible to the user.
  • Metadata is data about the data of the webpage, such as how it's displayed and the author. One reason it's useful for SEO, Which stands for Search Engine Optimization.
  • The body element represents the visible content shown to the user.
  • The Header and Body would be considered Elements. Things that go inside of them are considered Child Elements

CSS

  • CSS stands for Cascading Style Sheets, and is used to style and layout web pages
  • CSS can control colors, fonts, spacing, and positioning.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that allows you to upload, called push, and download, called pulls, and track changes in your codebase.
  • Git allows you to create branches for different features or bug fixes without affecting the main codebase.

JavaScript

  • JavaScript is a programming language used to add interactivity and functionality to websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
